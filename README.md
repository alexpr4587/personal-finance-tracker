# Personal Finance Tracker

A simple yet powerful **Python application** to track your **income and expenses**, visualize trends, and manage your personal budget.  
This project is designed to help you practice **Python basics, file handling, data analysis, and visualization**.

---

## 📌 Features
- Add and manage **income** and **expense** records.
- Categorize transactions (Food, Bills, Transport, etc.).
- Store data in CSV format.
- View **monthly summaries**.
- Generate **graphs of income vs expenses**.

---

## 🚀 Usage

1. **Example interaction**

- Select action (add/view/exit): add

- Enter a date (YYYY-MM-DD): 2025-08-15
- Enter 'income' or 'expense': income
- Enter expense amount: 230
- Enter category: Dividends
- Enter description: SCHD
- New data added successfully!
